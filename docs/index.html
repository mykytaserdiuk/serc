SER Programming Language

Simple interpreted programming language written in Go

Syntax Functions Structures Modules TODO Installation

Introduction

SER is a small interpreted programming language written in Go.

The goal of SER is to provide a simple syntax and lightweight runtime while still supporting functions, structures and modules.

Hello World

func main():
    print("Hello SER!\n")
end
Every program starts executing from the main() function.

Syntax

Comments

Comments start with ;. Everything after ; until the end of line is ignored. 

; this is a comment

func main():
    print("Hello")
end

Variables

Variables are created using def. 

func main():
    def name = "Oleg"
    def age = 20

    print(name)
end

Assignment

Existing variables can be changed without def. 

func main():
    def x = 0
    x = x + 1
end

Data Types

Type Description
Int Integer numbers
String Text values
Bool Boolean values
Null No value
Object Structures and complex values

Functions

Functions are declared using func. 

func hello():
    print("Hello")
end

Function arguments

func send(user):
    printf("%+v\n", user)
end

Return values

func getName():
    return "SER"
end

Functions without a return value return Null.

Conditions

Conditions use if and then. 

func main():
    if (5 + 2 <> 6) then
        print("true")
    end
end

Comparison operators

Operator Description
< Less than
> Greater than
<= Less or equal
>= Greater or equal
== Equal
<> Not equal

Loops

SER currently supports while loops. 

func main():
    def x = 0
    while(x < 10):
        printf("%+v\n", x)
        x = x + 1
    end
end

Structures

Structures allow creating custom objects.

Creating structure

struct User (
    first_name
    lastName
)

Creating object

func main():
    def user = User(
        first_name: "Oleg"
        lastName: "Vinnic"
    )
end

Accessing fields


user.first_name = "Daniel"
print(user.first_name)

Modules

Modules allow splitting programs into separate files.

Modules are stored inside the modules/ directory.

Example structure: 
project/
    main.ser

    modules/
        math.ser

Import module

use math
Example: 
; modules/math.ser

func add(a, b):
    return a + b
end
Usage: 
use math

func main():
    printf("%+v\n", math.add(1,5))
end

Module aliases

A module can have an alias. The alias becomes the name used inside the program. 

use math calculator

func main():
    calculator.add(1,5)
end

Built-in Functions

print

Prints values without formatting. 

print("Hello")

printf

Formatted output. Uses formatting similar to Go printf. 

printf("%s\n", "Hello")
printf("%+v\n", object)

Example Program

func main():
    def x = 0
    while(x < 10):
        printf("%+v\n", x)
        x = x + 1
    end
end

Installation

Build SER interpreter from source: 

git clone https://github.com/mykytaserdiuk/serc
cd serc
go build .

Run a SER program: 

./serc ./examples/loops.ser
